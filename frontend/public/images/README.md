# Images Directory

Place your images in the following structure:

## Home Page Images

### Hero Image
- `hero-home.jpg` - Main hero image for the home page (recommended size: 1920x600px)

### City Images
Place city images in the `cities/` subfolder:
- `cities/mumbai.jpg`
- `cities/bengaluru.jpg`
- `cities/ahmedabad.jpg`
- `cities/delhi.jpg`
- `cities/jaipur.jpg`
- `cities/hyderabad.jpg`
- `cities/chennai.jpg`
- `cities/kolkata.jpg`
- `cities/pune.jpg`
- `cities/surat.jpg`

All city images should be square format (recommended size: 400x400px or larger)

## About Page Images

### About Hero Image
- `about-hero.jpg` - Hero background image for the About page (recommended size: 1920x400px)
  - Should feature hands holding a miniature house with green trees
  - Will be blurred and darkened with overlay
